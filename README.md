# Movie Recommendation System

## Description
A content-based movie recommendation system built using Python and Scikit-Learn.

### Steps Followed in Developing the Movie Recommendation System

1. **Collected the Dataset**

   * Created a movie dataset containing movie titles and genres.
   * Stored the data in `movies.csv`.

2. **Data Preprocessing**

   * Loaded the dataset using Pandas.
   * Cleaned and organized movie information.

3. **Feature Extraction**

   * Used movie genres as features.
   * Converted text data into numerical vectors using CountVectorizer.

4. **Similarity Calculation**

   * Applied Cosine Similarity to measure similarity between movies.
   * Generated a similarity matrix for all movies.

5. **Recommendation Logic**

   * Developed a recommendation function.
   * Retrieved the most similar movies based on user input.

6. **User Interface Development**

   * Built an interactive web application using Streamlit.
   * Added a text input field and recommendation button.

7. **Poster Integration**

   * Added movie posters using a poster dictionary.
   * Displayed posters along with recommended movie titles.

8. **Testing and Debugging**

   * Tested recommendations for different movies.
   * Fixed issues related to incorrect recommendations and image paths.

9. **GitHub Integration**

   * Uploaded project files to GitHub.
   * Organized source code, dataset, and poster images.

10. **Deployment**

    * Deployed the application using Streamlit Community Cloud.
    * Made the project accessible online.

### Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* GitHub

### Algorithm Used

* Content-Based Filtering
* Cosine Similarity
