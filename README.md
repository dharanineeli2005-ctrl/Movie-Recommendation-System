# Movie Recommendation System

## Description
A content-based movie recommendation system built using Python and Scikit-Learn.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Streamlit

## Features
- Movie recommendations
- Similarity-based matching
- Simple user interface

## Files
- movies.csv
- ratings.csv
- recommendation.py
- app.py
